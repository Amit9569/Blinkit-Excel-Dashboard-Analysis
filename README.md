# 🛒 Blinkit Grocery Sales Analysis Dashboard (Excel Project)

## 📌 Project Overview

This project analyzes Blinkit grocery sales data using Microsoft Excel.
The goal of this project is to understand sales performance, customer
preferences, outlet performance, and product trends using an interactive
Excel dashboard.

The complete analysis workflow including dataset, pivot analysis, and
dashboard is available in a single Excel workbook.

---

## 🛠 Tools Used

- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Visualization

---

## 📊 Key Performance Indicators (KPIs)

### 💰 Total Sales
Overall revenue generated from all products.

### 📈 Average Sales
Average revenue generated per sale.

### ⭐ Average Rating
Customer satisfaction analysis based on ratings.

### 🛍 Number of Items
Total products available/sold.

---

## 📈 Dashboard Analysis

### 1. Total Sales by Fat Content
Analyzed customer preference between Low Fat and Regular products.

### 2. Sales by Item Type
Identified the best-performing grocery categories.

### 3. Fat Content by Outlet
Compared product sales across different outlets.

### 4. Sales by Outlet Establishment Year
Analyzed outlet growth and sales trends over time.

### 5. Sales by Outlet Size
Compared revenue contribution by outlet size.

### 6. Sales by Outlet Location
Identified location-wise sales performance.

### 7. Outlet Type Analysis
Compared sales, items, ratings, and performance across outlet types.

---

## 📌 Business Insights

- Found highest revenue generating product categories.
- Compared sales performance across outlet types.
- Analyzed customer preference based on product fat content.
- Evaluated outlet performance by size and location.
- Tracked sales trends over different years.
